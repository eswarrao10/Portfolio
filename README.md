## ✨ Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/eswarrao10/Portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
